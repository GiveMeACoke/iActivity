## I活动

**项目描述**：I活动是针对学校活动综测加分机制，开发一个校园活动平台。学生们可以在上面浏览，报名，参加校园活动，并得到相应的积分，学生干部可以在平台上申请活动，通过老师审核后进行发布，并对报名并参加搞活动的学生进行管理。
